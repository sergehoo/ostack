# Standards intégrés

Les profils de standards sont des règles versionnées et testables, pas des préférences cachées dans les prompts.

| Domaine | Profils initiaux |
|---|---|
| Backend | Django, FastAPI, Laravel, Spring Boot, ASP.NET, Go |
| Frontend | React, Next.js, Vue, Angular |
| Mobile | Flutter, React Native |
| Infrastructure | Docker, Kubernetes, Traefik, Nginx, Dokploy |
| CI/CD | GitHub Actions, GitLab CI, Jenkins |
| Données | PostgreSQL, MySQL, SQL Server, MongoDB, SQLite |

Chaque profil mature contiendra détection, règles, exceptions documentées, contrôles automatiques, exemples conformes et guide de migration.
